# Product Management Application

A modern, fully-featured task and product management application built with React 19, TypeScript, and Vite. This project combines user authentication with robust task and product management capabilities, featuring a responsive design that works seamlessly across mobile and desktop devices. The application is powered by Redux Toolkit for state management and integrates with the DummyJSON API for real-time data operations.

The application provides comprehensive CRUD operations for both tasks and products with an intuitive user interface built using Tailwind CSS. Users can register and log in securely, manage their personal task lists, and handle product inventory with features like search filters, status tracking, and categorization. All state changes are persisted in Redux with local storage for authentication tokens, ensuring a smooth user experience even after page refreshes.

To get started, clone the repository, run `npm install` to install dependencies, and then `npm run dev` to start the development server. For production deployment, use `npm run build` to create an optimized build. The application is fully type-safe with TypeScript, includes comprehensive error handling, and follows modern React best practices with component modularization and responsive design patterns.
